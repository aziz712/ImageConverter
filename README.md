# Image Converter Web App

A fast, modern, and fully responsive web application that allows users to upload, convert, preview, and download images in multiple formats (JPG, PNG, WebP, SVG, etc.).
Built with Next.js, TailwindCSS, and a lightweight Node.js/Express backend for image processing and compression.
